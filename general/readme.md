# Date Formats

Date formats can be a hell, as there are lots of them, which may even coexist in the same environment.

## Standards

| Standard | Format |
| :--- | :--- |
| ISO 8601 | yyyy-MM-dd'T'HH:mm:ss |
| POSIX/Unix time | Milliseconds |
